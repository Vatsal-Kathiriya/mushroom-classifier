About This Project

The Mushroom Classifier is a machine learning project designed to help identify whether mushrooms are edible or poisonous based on their physical characteristics.

How It Works

This classifier uses a Logistic Regression model trained on a dataset of mushroom characteristics. Logistic Regression is ideal for binary classification tasks like determining edibility.

The model analyzes features like cap shape, color, gill appearance, and habitat to make predictions, with coefficients indicating how each feature influences the prediction.
